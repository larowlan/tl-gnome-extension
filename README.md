# TL Gnome Extension

## Description
Custom Gnome extension to show your TL status in the MessageTray

## Installation
Clone this repo to `~/.local/share/gnome-shell/extensions/tl@l.pnx.me`
Enable this extension with `gnome-extensions enable tl@l.pnx.me`
Restart Gnome
